### How to filter specific logs?

If you click on the **Filter entries** box, you can filter logs based on **user**, **decision** or **resource**.

![Audit Log permitted hover Screenshot](/img/screenshots/audit-log-filter.png)

You may also filter according to **time**. To filter by time you can use the **date filter box** on the left and choose the desired timespan.

![Audit Log permitted hover Screenshot](/img/screenshots/audit-log-time-filter.png)

:::info
Below you can see that we filtered all the `denied` decisions on the `board` resource that were made in the last month.
:::
