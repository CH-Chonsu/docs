# Customizing User Data

<span className="subtitle-custom">
	Customize user data for more or less user information.
</span>
<br />
<br />
Our permit element allows you to customize the user data that is displayed. You can
choose to display either the user's email, full name, or both. It is not possible
to display no user data.
