# getAssignedRoles

Get all the assigned roles for a specific user.

### Parameters

`user` - The id of the role. This can also be the role key.<br/>
`tenant` - **optional** - The id of the tenant or the tenant key. If no is tenant provided, all tenants will fetch.<br/>

### Payload

```python
{
	"user": "user",
	"tenant": "tenant"
}
```

### Implementation

```python
roles = await permit.api.get_assigned_roles(user, tenant)
```
