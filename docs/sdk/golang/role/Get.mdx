# Get / GetByKey

Get a single tenant role by Key, if such role exists.

### Parameters
`ctx` -  The context of the request.<br/>
`roleKey` - The key of the role.<br/>

### Implementation

```golang
role, err := Permit.Api.Roles.Get(ctx, "role-key")
```

# GetById
Get a single tenant role by ID, if such role exists.

### Parameters

`roleId` - The id of the role. This can also be the role key.<br/>

### Implementation

```golang
    role, err := Permit.Api.Roles.GetById(ctx, uuid.String())
```
