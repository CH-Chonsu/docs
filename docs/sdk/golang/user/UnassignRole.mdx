# UnassignRole

Unassign a role from a specific user.

### Parameters
`ctx` -  The context of the request.<br/>
`unassignedRole` - An object that contains the information about the role. The payload is defined below.<br/>

### Payload

`ctx` -  The context of the request.<br/>
`userKey` - The key of the user to assign the role to.<br/>
`roleKey` - The key of the role to assign to the user.<br/>
`tenantKey` - The key of the user's tenant.<br/>

### Implementation

```go
	user, err := Permit.Api.Users.UnasignRole(ctx, "user-key", "role-key", "tenant-key")
```
