# Get / GetByKey

Get the specificed user and their information.

### Parameters
`ctx` -  The context of the request.<br/>
`userKey` - The key of the user. This is the unique key of the user.<br/>

### Implementation

```go
user, err := Permit.Api.Users.Get(ctx, "user-key")
```

# GetById

Get the specificed user and their information.

### Parameters
`ctx` -  The context of the request.
`userId` - The ID of the user. This is the unique key of the user.<br/>

### Implementation

```go
user, err := Permit.Api.Users.Get(ctx, uuid.String())
```
