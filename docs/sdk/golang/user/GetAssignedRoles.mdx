# GetAssignedRoles

Get all the assigned roles for a specific user.

### Parameters

`ctx` -  The context of the request.<br/>
`userKey` - The key of the user to assign the role to.<br/>
`tenantKey` - The key of the user's tenant.<br/>
`page` - The page number to return.<br/>
`perPage` - The number of results to return per page.<br/>

### Implementation

```golang
	rolesAssignments, err := Permit.Api.Users.GetAssignedRoles(ctx, "role-key", "default",1,10)
```
