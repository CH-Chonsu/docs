# Update

Updates the tenant.

### Parameters
`ctx` -  The context of the request.<br/>
`tenantKey` - The id of the tenant. This can also be the tenant key.<br/>
`tenantUpdate` - An object that contains the information about the tenant. The payload is defined below.<br/>

### Payload

`Name` - **optional** - A descriptive name for the tenant.<br/>
`Description` - **optional** - A longer description of the tenant.<br/>
`Attributes` - **optional** - Arbitraty tenant attributes that will be used to enforce attribute-based access control policies.<br/>

```go
tenantUpdate := models.NewUpdateTenant()
tenantUpdate.SetName("New Tenant Name")
```

### Implementation

```go
tenant, err := Permit.Api.Tenants.Update(ctx, "tenant-key", *tenantUpdate)
```
