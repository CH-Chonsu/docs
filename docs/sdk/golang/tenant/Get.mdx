# Get / GetByKey

Fetch the tenant and the tenant information.

### Parameters
`ctx` -  The context of the request.<br/>
`tenantKey` - The Key of the tenant.<br/>

### Implementation

```golang
tenant, err := Permit.Api.Tenants.Get(ctx, "tenantKey")
```


# GetById
### Parameters
`ctx` -  The context of the request.<br/>
`tenantId` - The id of the tenant. This can also be the tenant key.<br/>

### Implementation

```golang
tenant, err := Permit.Api.Tenants.GetById(ctx, uuid.String())
```
