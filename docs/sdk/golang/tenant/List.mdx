# List

Fetch a list of the available tenants.

### Parameters

`ctx` -  The context of the request.<br/>
`page` - The page number of the results to fetch, starting at 1.<br/>
`perPage` - The number of results to fetch per page.<br/>

### Implementation

```go
tenants, err := Permit.Api.Tenants.List(ctx, 1, 10)
```
