# assignRole

Assign a role to a specific user.

### Parameters

`user` - The user the role will be assigned to (accepts either the user id or the user key).<br/>
`role` - The role that will be assigned (accepts either the role id or the role key).<br/>
`tenant` - The tenant the role is associated with (accepts either the tenant id or the tenant key).<br/>

### Payload

```python
{
	"role": "role",
	"tenant": "tenant",
	"user": "user"
}
```

### Implementation

```python
response = await permit.api.assign_role(user,role,tenant)
```
