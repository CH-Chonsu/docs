To get the available roles, you can use the [API](https://api.permit.io/redoc#tag/Roles/operation/list_roles).

If you send a request to `https://api.permit.io/api/v1/roles` with your webapp bearer token you will get the response:

```json
{
	"url": "string",
	"data": [
		{
			"name": "string",
			"description": "string",
			"attributes": {},
			"settings": {},
			"id": "497f6eca-6276-4993-bfeb-53cbbbba6f08",
			"permissions": [],
			"isBuiltIn": false
		}
	]
}
```

We will update the python command once a new version of this SDK is released.

If you need it for your project, please [let us know.](mailto:help@permit.io)
