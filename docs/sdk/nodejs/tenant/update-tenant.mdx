# updateTenant

Updates the tenant.

### Parameters

`tenantId` - The ID of the tenant. This can also be the tenant name. <br/><br/>
`tenant` - An object that contains the information about the tenant. The payload is defined below.

### Payload

`name` - **optional** - A descriptive name for the tenant.<br/><br/>
`description` - **optional** - A longer description of the tenant.<br/><br/>
`attributes` - **optional** - Arbitraty tenant attributes that will be used to enforce attribute-based access control policies.<br/>

```json
{
	"name": "name",
	"description": "description",
	"attributes": {}
}
```

### Implementation

```js
const response = await permit.api.updateTenant(tenantId, ${JSON.stringify(tenant)});
```
