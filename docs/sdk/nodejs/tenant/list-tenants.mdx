# listTenants

Fetch a list of the available tenants.

### Parameters

`page` - The page number of the results to fetch. <br/>

### Required Payload

```json
{
	"name": "name",
	"description": "description",
	"attributes": {}
}
```

### Implementation

```js
const tenantList = await permit.api.listTenants(page);
```
