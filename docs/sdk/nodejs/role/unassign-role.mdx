# unassignRole

Unassign a role from a specific user.

### Parameters

`unassignedRole` - An object that contains the information about the role. The payload is defined below.<br/>

### Payload

`role` - The role that will be unassigned (accepts either the role id or the role key).<br/><br/>
`tenant` - The tenant the role is unassociated with (accepts either the tenant id or the tenant key).<br/><br/>
`user` - The user the role will be unassigned to (accepts either the user id or the user key).<br/><br/>

```json
{
	"role": "role",
	"tenant": "tenant",
	"user": "user"
}
```

### Implementation

```js
const response = await permit.api.unassignRole(${JSON.stringify(unassignedRole)});
```
