# createRole

Create a new role.

### Parameters

`roleId` - RoleCreate <br/> <br/>

### Payload

```json
{
	"key": "editor",
	"name": "Editor",
	"description": "the editor role can read and write to documents",
	"permissions": ["document:write"],
	"extends": ["viewer"]
}
```

### Implementation

```js
const response = await permit.api.createRole(${JSON.stringify(role)});
```

### Response Example

```json
{}
```
