# getAssignedRoles

Get all the assigned roles for a specific user.

### Parameters

`user` - The ID of the role. This can also be the role name. <br/><br />
`tenant` - **optional** - The ID of the tenant or the tenant name. If no is tenant provided, all tenants will fetch.

### Payload

```json
{
	"user": "user",
	"tenant": "tenant"
}
```

### Implementation

```js
const roles = await permit.api.getAssignedRoles(user, tenant);
```
