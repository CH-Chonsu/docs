# getAssignedRoles

Get all the assigned roles for a specific user.

### Parameters

`user` - The ID of the role. This can also be the role name. <br/>
`tenant` - Optional - if no tenant provided, all tenants will fetch.

### Implementation

```js
const roles = await permit.api.getAssignedRoles(user, tenant);
```
