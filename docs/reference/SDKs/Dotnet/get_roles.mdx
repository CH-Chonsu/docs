title: GetRoles
---

To get the available roles, you can use the following functions

The different between those 2 functions is that the Cache function calls the local sidecar without latency.

And the API one calls Permit cloud.

So if you call this function many times or in time critical code, you should use the Cache function.



```csharp
Role[] roles = await permit.Cache.GetRoles();
// returns your org roles with list of SyncedRole type
// [{
//         id: string,
//         name: string,
//         description: string,
//         permissions: string[],
//         metadata: Dictionary<string, object>,
//     },
//    ...
// }


Role[] roles = await permit.Api.GetRoles();
// returns your org roles with list of Role type
// [{
//         id: string,
//         name: string,
//         description: string,
//         tenantId: string,
//     },
//    ...
// }


```
