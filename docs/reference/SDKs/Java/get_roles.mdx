---

To get the available roles, you can use the following functions

The different between those 2 functions is that the Cache function calls the local sidecar without latency.

And the API one calls Permit cloud.

So if you call this function many times or in time critical code, you should use the Cache function.



```java
RoleAssignmentList roles = permit.Api.getAssignedRolesInAllTenants();
// returns your org roles with list of RoleAssignmentList type
// [{
//         id: string,
//         name: string,
//         description: string,
//         tenantId: string,
//     },
//    ...
// }


```
