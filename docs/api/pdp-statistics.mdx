# PDP Statistics

Permit.io PDPs report some basic statistics to the API, and you can query that through the statistics API.
API docs are available at [https://pdp-statistics.api.permit.io/v2/redoc](https://pdp-statistics.api.permit.io/v2/redoc).

## Using the PDP Statistics API
To get PDP statistics, simply use your existing API key to perform a GET request to 
https://pdp-statistics.api.permit.io/v2/statistics/{project}/{environment}/pdps

As with other APIs, the project and environment can be either IDs or keys.
